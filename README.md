# React demo project

### Setup

 - Install node v5.1.0
 - `npm install`


### Run web app

 - `npm run build`
 - `npm run serve`
 - Go to [http://localhost:3000](http://localhost:3000)


### Run tests

 - `npm test`
 - `npm run t src/tests/WeatherStore-test.jsx`
 - `npm run webtest`
 - `npm run tdd`
